# Website template configured with semantic-ui

This is just a template to build HTML + JS simple websites, without frameworks.

## Install

run the following command to install all dependencies:

```bash
npm install
npm install -g gulp
cd semantic
gulp build

```

## Build

```bash
npm run build
```

## Run

```bash
npm run watch
```

## Fast build and run command

```bash
npm start
```
